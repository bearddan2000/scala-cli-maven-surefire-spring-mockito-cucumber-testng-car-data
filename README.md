# scala-cli-maven-surefire-spring-mockito-cucumber-testng-car-data

## Description
A POC for maven app using testng
mockito, and cucumber framework
 with surefire plugin.

## Tech stack
- scala
- maven
	- spring
  - testng
  - surefire
  - cucumber
	- mockito


## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
